# FitGirl Downloader Pro (FDP)

![FDP Hero](https://images.unsplash.com/photo-1542751371-adc38448a05e?auto=format&fit=crop&q=80&w=1200)

**FitGirl Downloader Pro** is a high-performance, modern web utility designed to streamline the retrieval and management of game repack downloads. Built for the gaming community, it bridges the gap between repack site navigation and high-speed download managers.

## 🚀 Key Features

- **Link Extractions**: Automatically resolve and structure multi-part download links from FitGirl repack pages.
- **Mirror Support**: Intelligent detection of high-speed mirrors (FuckingFast).
- **Smart Queue Management**: Add hundreds of part files to a download queue with a single click.
- **Modern UI/UX**: A sleek, dark-themed interface built with a focus on usability and precision.
- **External Export**: One-click "Copy All" functionality to export links directly to your favorite download manager (IDM, JDownloader2, etc.).
- **Real-time Monitoring**: Track download progress, speed, and status with interactive visual components.

## 🛠️ Built With

- **Framework**: [React 18](https://reactjs.org/) with [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Motion (Framer Motion)](https://motion.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **State Management**: React Hooks & Context

## 🏁 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Rubayet123/fitgirl-downloader-pro.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd fitgirl-downloader-pro
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```

5. **Build for production**
   ```bash
   npm run build
   ```

## 📖 Usage Guide

1. **Fetch**: Paste a valid repack page URL or direct FuckingFast link into the fetcher.
2. **Resolve**: Wait for FDP to scan for available mirrors and part lists.
3. **Queue**: Use "Add All to Queue" or individual "+" buttons to prepare your downloads.
4. **Download**: Start individual items or copy links to use in external tools.
